# Change: Add Custom Backgrounds

## Why
Cards目前仅支持固定渐变背景，无法满足用户想用照片/自定义纹理提升差异化的需求。外链图片易触发 html2canvas 跨域污染，未经压缩的大图也会导致导出失败或存储爆炸，需要一个可控的本地上传方案。

## What Changes
- 支持本地上传图片作为卡片背景，统一压缩为安全的 data URL，避免跨域问题。
- 提供遮罩强度滑块（默认约 40% 黑色遮罩），保证文字对比度，并允许用户调整。
- 使用 `<img object-fit: cover>` 渲染背景，并在导出前等待图片加载，失败回退默认渐变。
- 增加文件类型/大小校验与压缩（最长边 ≤1200px，JPEG 质量约 0.7）以控制性能和存储。
- 仅存储一张当前背景（内存/可选 localStorage），避免配额超限；提供清除/重置操作。

## Impact
- 影响能力：新建自定义背景相关能力。
- 影响代码：`components/Card.tsx`、`components/Editor` 背景上传 UI、压缩/校验工具、导出逻辑等待图片加载、主题/遮罩配置常量。
